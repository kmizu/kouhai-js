関数 買い物リスト管理():
    リスト = []

    繰り返し:
        表示 "1: 追加, 2: 表示, 3: 終了"
        選択 = 入力()

        もし 選択 == "1":
            商品 = 入力("商品名: ")
            リスト.追加(商品)
        そうでなければ 選択 == "2":
            表示 "買い物リスト:"
            各要素 商品 の リスト:
                表示 "- {商品}"
        そうでなければ:
            終了
