# 第3話 バレたブックマーク

## 10月5日（水）午前7時30分

「隆弘くん、今日もプログラミング教えてくれる？」

マンションのロビーで、美久が期待に満ちた瞳で聞いてくる。

「もちろん。今日はHTMLの実践編だ」

僕は胸を張って答えた。プログラミングを教えることが、こんなに楽しいなんて思わなかった。

（でも、昨日のlovePointの件があるからな……）

ちょっと気まずい雰囲気が残っているかもしれない。でも、美久は何事もなかったかのように振る舞っている。

「楽しみ！昨日の夜、ずっと復習してたんだ」

「え、本当に？」

「うん。変数の宣言とか、if文の書き方とか。完璧！」

得意げに胸を張る美久。その姿が可愛くて、つい笑ってしまう。

「テストしてみようか？」

「望むところです！」

こんな風に、プログラミングの話で盛り上がれる相手がいるなんて。しかも、それが美久だなんて。

◇◇◇◇

## 午後12時20分　屋上

「嵐山、最近楽しそうだな」

昼休み、屋上でパンを食べていると、プログラミング部の同期、田中が話しかけてきた。

「そうか？」

「だって、文芸部の河内さんと一緒にゲーム作ってるんだろ？羨ましいな〜」

田中がニヤニヤしながら肘で小突いてくる。

「別に、ただの合同企画だよ」

「へー、じゃあなんで顔が赤いんだ？」

（バレバレか……）

僕は咳払いをして、話題を変えようとした。

「それより、今日の部活どうする？」

「お、逃げたな。まあいいや。今日は新入部員の勧誘ポスター作らないと」

そうだった。文化祭の展示があるから、もっと部員が欲しいところだ。

「ゲーム制作の実演とかどう？」

「それいいな！河内さんも一緒に？」

「……考えておく」

田中の提案は悪くない。でも、美久と一緒に実演となると、また違った緊張感がありそうだ。

◇◇◇◇

## 午後4時30分　プログラミング部室

「さて、今日はHTMLでゲーム画面を作ってみよう」

僕はパソコンを起動し、Visual Studio Codeを立ち上げた。

「わあ、本格的！」

美久が目を輝かせる。

「まずは、ゲームの基本的なレイアウトから」

僕は慣れた手つきでコードを打ち始めた。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>私たちのノベルゲーム</title>
</head>
<body>
    <div id="game-container">
        <div id="character-area">
            <!-- キャラクターの立ち絵 -->
        </div>
        <div id="text-box">
            <!-- セリフ表示エリア -->
        </div>
        <div id="choices">
            <!-- 選択肢 -->
        </div>
    </div>
</body>
</html>
```

「これが基本構造。divタグで画面を区切って……」

説明しながら、僕は画面を分割していく。そして、実際のゲーム画面に近づけるために、もう少し詳細を追加していく。

「キャラクターの立ち絵はimg タグで表示して、テキストボックスは……」

その時だった。

ブラウザを開こうとして、誤ってブックマークバーを表示してしまった。

そこには――

『夏色の約束 -Summer Promise-』
『月光のセレナーデ』
『桜舞う頃に、君と』

明らかにギャルゲーのタイトルが並んでいる。

時間が止まった。

美久の視線が、画面に釘付けになっている。

（終わった……）

顔から血の気が引いていくのを感じる。慌ててブックマークバーを閉じようとするが、もう遅い。

「あの……先輩？」

美久の声が、妙に落ち着いている。

「こ、これは、その……ゲーム制作の参考資料として……」

苦しい言い訳だと自分でも分かっている。

美久がゆっくりと口を開いた。

「『夏色の約束』って……」

（知ってるの！？）

「去年の夏に発売された、学園恋愛ものですよね」

僕は凍りついた。まさか美久がギャルゲーのタイトルを知っているなんて。

「い、いや、ただの偶然……」

「私、そのゲーム……」

美久が何か言いかけて、急に口をつぐんだ。頬が赤くなっている。

重い沈黙が部室を包む。

お互いに目を合わせられない。

でも、なんだろう。この気まずさの中に、不思議な共犯意識みたいなものを感じる。

「あの、先輩」

「な、なんだ？」

「実は私も……いえ、なんでもないです」

美久は首を振って、また画面に視線を戻した。

「それより、このレイアウト、すごく本格的ですね」

話題を変えてくれた美久に、僕は心の中で感謝した。

「あ、ああ。テキストボックスの配置は大事だからね」

僕は震える手でコードを打ち続ける。

```html
<div id="text-box">
    <p id="character-name">？？？</p>
    <p id="dialogue">ここにセリフが表示されます。</p>
</div>
```

「キャラクター名と、セリフを分けて表示するんだ」

「なるほど……」

美久は真剣にメモを取っている。でも、その頬はまだ少し赤い。

（もしかして、美久も……？）

いや、考えすぎだ。でも、さっきの反応は明らかに――

「先輩、選択肢のボタンはどうやって作るんですか？」

美久の質問で、僕は我に返った。

「ああ、buttonタグを使うんだ」

```html
<div id="choices">
    <button class="choice-btn">選択肢1</button>
    <button class="choice-btn">選択肢2</button>
</div>
```

「これで、クリックできる選択肢が作れる」

「すごい！本当にゲームみたい！」

美久の素直な反応に、少しずつ緊張が解けていく。

◇◇◇◇

## 午後6時　プログラミング部室

「今日はここまでにしようか」

2時間近く集中して作業をして、基本的なHTMLの構造が完成した。

「はい……あの、先輩」

片付けをしながら、美久が声をかけてきた。

「さっきのブックマークの件ですけど」

（やっぱり触れるのか……）

「私、何も見てませんから」

美久は優しく微笑んだ。

「それに、ゲーム制作の参考資料は大事ですよね」

その言葉に、なぜか違和感を覚える。まるで、自分に言い聞かせているような……

「美久？」

「なんでもないです！それより、明日はCSSですよね？」

話題を逸らす美久。でも、その表情には、昨日のlovePointの時と同じような、何か隠しているような雰囲気がある。

「ああ、そうだね。画面を綺麗にする方法を教えるよ」

「楽しみです！」

◇◇◇◇

## 午後7時30分　自室

部屋に戻って、僕は今日の出来事を振り返っていた。

ギャルゲーのブックマークがバレた。これは相当恥ずかしい。

でも、美久の反応が気になる。

『夏色の約束』のことを知っていた。それも、発売時期まで正確に。

普通の女子高生が知っているような情報じゃない。

まさか……

僕はスマホを手に取り、『夏色の約束』の公式サイトを開いた。

そして、ふと思い出す。このゲーム、男性向けだけじゃなくて、女性ファンも多いことで有名だった。シナリオの評価が高くて、恋愛描写が丁寧で……

（もしかして、美久も？）

いや、でも、決めつけるのは良くない。

でも、もし、本当に美久もゲームが好きだとしたら……

窓の外を見ると、隣の部屋の明かりがついている。

美久は今、何をしているんだろう。

もしかして、僕と同じように、今日のことを考えているのかもしれない。

明日、さりげなく聞いてみようか。

いや、でも、どう聞けばいいんだ。

「美久、実はゲーム好き？」なんて直球すぎる。

でも、このままモヤモヤしているのも……

僕は、ため息をついた。

プログラミングを教えるだけのはずだったのに、なんだか関係が複雑になってきている。

でも、不思議と嫌じゃない。

むしろ、美久との共通点が見つかるかもしれないという期待の方が大きい。

明日は、CSSを教える。

そして、もしチャンスがあれば、ゲームの話も……

そんなことを考えながら、僕は明日の授業の準備を始めた。

画面には、今日作ったHTMLが表示されている。

『私たちのノベルゲーム』

このタイトルが、なんだか特別な意味を持ち始めている気がした。
