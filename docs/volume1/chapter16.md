# 第16話 文化祭実行委員との交渉

## 10月24日（火）午後4時

「嵐山、ちょっといいか」

文化祭実行委員長の3年生、山田先輩が部室に入ってきた。

「どうしました？」

「展示場所の件なんだが」

山田先輩の表情が険しい。

嫌な予感がする。

「実は、体育館の展示スペースが予定より狭くなってな」

「え？」

「設備の関係で、各団体のスペースを縮小せざるを得ない」

まさかの事態に、頭が真っ白になる。

「でも、僕たちはゲームの展示で、PCを複数台設置する必要が……」

「分かってる。だが、他の団体も同じ状況だ」

山田先輩も困った顔をしている。

「何か、解決策はありませんか？」

僕は必死に食い下がった。

◇◇◇◇

## 午後4時30分　緊急ミーティング

「大変なことになった」

部員たちを集めて、状況を説明する。

「スペースが狭くなるって、どれくらい？」

田中が心配そうに聞く。

「半分以下になるかもしれない」

その言葉に、みんなの顔が青ざめた。

「そんな……せっかくここまで作ったのに」

美久が肩を落とす。

「諦めるのは早い。何か方法があるはずだ」

僕は部員たちを励ました。

でも、正直、僕も不安でいっぱいだった。

◇◇◇◇

## 午後5時　美久との作戦会議

「隆弘くん、何かいい案はない？」

美久が心配そうに聞いてくる。

「考えているんだけど……」

頭を抱える僕。

「そうだ！」

美久が突然声を上げた。

「どうした？」

「展示方法を工夫すればいいんじゃない？」

「展示方法？」

「例えば、Webブラウザで動くようにすれば、来場者のスマホでも遊べる」

美久の提案に、目から鱗が落ちた。

「それだ！」

◇◇◇◇

## 午後5時30分　技術的検証

「Webアプリとして公開すれば、展示スペースは最小限で済む」

僕はすぐに実装方法を検討し始めた。

```javascript
// PWA（Progressive Web App）として実装
const serviceWorkerRegistration = async () => {
    if ('serviceWorker' in navigator) {
        try {
            const registration = await navigator.serviceWorker.register('/sw.js');
            console.log('Service Worker登録成功:', registration);
        } catch (error) {
            console.error('Service Worker登録失敗:', error);
        }
    }
};

// QRコードでアクセスできるようにする
const generateQRCode = (url) => {
    const qr = new QRious({
        element: document.getElementById('qrcode'),
        value: url,
        size: 200
    });
    return qr;
};
```

「これなら、QRコードを掲示して、来場者が自分のスマホでプレイできる」

「すごい！これなら省スペースで大勢が遊べる！」

美久が興奮した声を上げる。

◇◇◇◇

## 午後6時　実行委員会への再提案

「山田先輩、提案があります」

僕は実行委員会室を訪れた。

美久も一緒だ。

「なんだ？」

「展示方法を変更することで、スペース問題を解決できます」

僕は準備した資料を見せながら説明した。

- Webアプリとして公開
- QRコード掲示で来場者がアクセス
- 必要なのは説明パネルとデモ機1〜2台のみ
- 同時に多数がプレイ可能

「これは面白い」

山田先輩の表情が和らぐ。

「技術的に可能なのか？」

「はい。すでにプロトタイプも作りました」

美久がタブレットでデモを見せる。

「素晴らしい。これなら他の団体の参考にもなる」

◇◇◇◇

## 午後6時45分　新たな課題

「でも、一つ問題が」

山田先輩が指摘する。

「ネットワーク環境だ。当日、大勢がアクセスしたら……」

確かに、それは盲点だった。

「学校のWi-Fiでは対応しきれないかもしれません」

「そうだな」

また壁にぶつかった。

でも、美久が言った。

「オフラインでも動くようにすればいいんです」

「オフライン？」

「PWAの機能を使えば、一度アクセスしたらオフラインでも動きます」

美久の知識に驚く。

いつの間に、そんなことまで勉強したんだ。

◇◇◇◇

## 午後7時30分　実装作業

```javascript
// Service Workerでオフライン対応
self.addEventListener('install', (event) => {
    event.waitUntil(
        caches.open('game-v1').then((cache) => {
            return cache.addAll([
                '/',
                '/index.html',
                '/game.js',
                '/style.css',
                '/images/characters/',
                '/images/backgrounds/',
                '/sounds/'
            ]);
        })
    );
});

self.addEventListener('fetch', (event) => {
    event.respondWith(
        caches.match(event.request).then((response) => {
            return response || fetch(event.request);
        })
    );
});
```

「これで、一度読み込めばオフラインでも動く」

「隆弘くん、天才！」

美久が目を輝かせる。

「美久のアイデアのおかげだよ」

二人で顔を見合わせて笑う。

ピンチがチャンスに変わった瞬間だった。

◇◇◇◇

## 午後8時　承認

「嵐山、河内、素晴らしい提案だ」

山田先輩から正式な承認が下りた。

「むしろ、特別展示枠として、入り口近くのスペースを用意しよう」

「本当ですか！」

思わぬ好条件に、美久と顔を見合わせる。

「最新技術を使った展示として、注目を集めるだろう」

「ありがとうございます！」

二人で深々と頭を下げた。

◇◇◇◇

## 午後8時30分　帰り道

「やったね、隆弘くん！」

美久が飛び跳ねるように喜ぶ。

「美久のおかげだよ」

「ううん、二人で解決したんだよ」

そう言って、美久は僕の腕に手を回してきた。

「美久？」

「嬉しくて」

僕も美久の手を握り返す。

「文化祭、成功させよう」

「うん！」

月明かりの下、手を繋いで歩く。

困難を二人で乗り越えたことで、絆がより深まった気がする。

「ねえ、隆弘くん」

「ん？」

「完成まで、あと少しだね」

美久の言葉に、約束を思い出す。

ゲームが完成したら、伝えたいことがある。

その時が、もうすぐそこまで来ている。

「うん、あと少しだ」

プログラミングの技術だけでなく、問題解決能力も身についた。

そして何より、美久との絆が、どんな困難も乗り越える力になることを知った。

文化祭まで、あと2週間。

最高の展示にしよう。
