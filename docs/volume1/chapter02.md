# 第2話 合同企画の始まり

## 10月4日（火）午前7時20分

朝。隆弘くんの部屋のドアをノックする。

いつもの時間。いつもの場所。でも、昨日からすべてが違って見える。

「おはよう、美久」

ドアを開けた隆弘くんの髪が、少し寝癖でくしゃっとしている。その無防備な姿に、私の心臓がきゅんと鳴った。

（ダメダメ、落ち着いて）

「おはよう、隆弘くん」

家での呼び方。学校に着いたら「先輩」になる。この切り替えが、なんだか秘密を共有しているみたいで、ドキドキする。

エレベーターの中、隆弘くんの横顔を盗み見る。真剣な表情でスマホを見ている。きっと、プログラミングの記事か何かを読んでいるんだろう。

その横顔が、朝の光に照らされて、まるで画面から出てきたゲームのキャラクターみたい。

（あ、また変なこと考えてる）

私は慌てて視線を逸らした。でも、狭いエレベーターの中では、隆弘くんの存在から逃げられない。シャンプーの香りに混じって、ほんのり香る柔軟剤の匂い。こんなに近くにいるのに、触れられない距離。

◇◇◇◇

## 午後0時15分　学食

「美久ちゃん、最近幸せそうだね」

同じ文芸部の友達、さやかがニヤニヤしながら言う。

「そ、そんなことないよ」

「だって、プログラミング部の嵐山先輩と一緒でしょ？」

嵐山先輩。学校での隆弘くんの呼び方。その名前を聞くだけで、顔が熱くなる。

「ただの合同企画だから」

「へー、じゃあなんでそんなに顔が赤いの？」

さやかの追及に、私は必死に否定する。でも、心の中では認めざるを得ない。

隆弘くんと一緒にゲームを作れることが、嬉しくて仕方ない。

「それより、今日の放課後、みんなで企画会議でしょ？」

話題を変えようとする私に、さやかは意味深な笑みを浮かべた。

「うん。嵐山先輩も来るんだよね？楽しみー」

（さやかまで……）

でも、正直なところ、私も楽しみで仕方なかった。

◇◇◇◇

## 午後4時30分　文芸部室

「みんな、紹介します。プログラミング部部長の嵐山隆弘先輩です」

私の声が少し上ずってしまった。でも、誰も気づいていない……はず。

部室に入ってきた隆弘くん……いや、隆弘先輩は、いつもよりちょっと緊張した表情をしている。その姿が新鮮で、つい見とれてしまう。

「初めまして。今日から一緒にゲームを作ることになりました。よろしくお願いします」

丁寧に頭を下げる隆弘先輩。その真摯な態度に、部員のみんなも好印象を持ったみたい。

企画会議が始まると、みんな思い思いのアイデアを出し始めた。

「私は、学園もののラブストーリーがいいと思います」

「ファンタジー要素も入れたい！」

「ミステリーもいいんじゃない？」

活発な議論を聞きながら、私は自分の本当の気持ちを言い出せずにいた。

本当は、乙女ゲームみたいなのを作りたい。選択肢によって攻略キャラとの関係が変わって、ドキドキするイベントがあって、最後には素敵な告白シーンがあって……。

でも、そんなこと言ったら、隆弘先輩に引かれちゃうかもしれない。

「美久はどんなのがいいと思う？」

隆弘先輩の声で、現実に引き戻される。みんなの視線が私に集まる。

心臓がドキドキする。でも、これは言わなきゃ。

「私は……選択肢で物語が変わる、恋愛シミュレーションみたいなのがいいかなって」

言った。言っちゃった。

隆弘先輩の表情が、一瞬驚いたように見えた。

（あ、やっぱり変だったかな）

後悔が押し寄せる。でも、次の瞬間。

「いいね！攻略キャラとか作ろうよ」

「主人公は女の子？男の子？」

みんなが賛同してくれた。そして、隆弘先輩も。

「プログラミング的には問題ありません。選択肢による分岐も、JavaScriptで実装できます」

その言葉に、私の心に花が咲いたような気がした。

◇◇◇◇

## 午後5時30分　プログラミング部室

文芸部の会議が終わって、隆弘先輩と二人きり。

この時間が、私にとって一番特別な時間。

「それで、今日は変数について教えるって言ってたよね」

「はい！予習してきました」

昨夜、必死に調べたノートを見せる。隆弘先輩に褒められたくて、頑張った成果。

「すごいな、ここまで調べたのか」

「だって、早く理解したいから」

本当は、隆弘先輩と同じ世界を見たいから。プログラミングの楽しさを、一緒に感じたいから。

隆弘先輩がコードを打ち始める。その指の動きを、じっと見つめる。

```javascript
let playerName = "美久";
let affection = 0;

console.log(playerName + "の好感度：" + affection);
```

私の名前が画面に表示される。それだけで、なんだか特別な気分。

「これって、ゲームでよくある……」

「そう、好感度システムだね」

好感度。その言葉に、私の頭の中で妄想が広がる。

もし現実にも好感度が見えたら、隆弘先輩の私への好感度はどのくらいなんだろう。幼馴染ポイントはきっと高いけど、恋愛対象としては……。

「じゃあ、選択肢によって好感度が変わるようにもできるんですか？」

質問しながら、心の中では別のことを考えている。

隆弘先輩がさらにコードを追加する。if文の説明を聞きながら、私は思い切って自分でもコードを打ってみることにした。

```javascript
let heroName = "隆弘先輩";
let lovePoint = 100;
```

打った瞬間、我に返る。

（何してるの私！）

「こ、これは……」

隆弘先輩の反応に、顔が真っ赤になる。恥ずかしくて死にそう。

「あ！ち、違うんです！これはただの例で……」

必死に言い訳する私。でも、本心がバレバレだ。

「い、いや、例としては分かりやすいよ。うん」

隆弘先輩の優しいフォローが、逆に恥ずかしさを増幅させる。

でも、ちょっとだけ、ほんのちょっとだけ、隆弘先輩の頬も赤くなってるような……気のせいかな。

◇◇◇◇

## 午後7時　マンションへの帰り道

暗くなった道を、隆弘先輩と二人で歩く。

学校では「先輩」と「後輩」。でも、この帰り道では、いつもの距離感に戻れる。それが嬉しくて、でも同時に切ない。

「あの、先輩」

「ん？」

「本当は……」

言いかけて、止まる。

本当は、乙女ゲームが大好きなこと。本当は、隆弘先輩ともっと一緒にいたいこと。本当は、lovePointは冗談じゃなくて本気だってこと。

でも、言えない。

「いえ、なんでもないです」

隆弘先輩は不思議そうな顔をしたけど、それ以上聞いてこなかった。その優しさが、また胸をきゅっとさせる。

マンションのエレベーターの中、また二人きり。朝と同じ空間なのに、雰囲気が全然違う。

プログラミングを教えてもらった後の、この特別な空気感。共通の秘密を持った共犯者みたいな、そんな感覚。

「先輩、今日のlovePointの件、本当にごめんなさい」

別れ際、もう一度謝る。

「だから、気にしなくていいって」

「でも……」

本当は、謝りたいんじゃない。あれが本心だって、伝えたい。でも、そんな勇気はない。

「おやすみなさい」

ドアを閉めて、部屋に入る。すぐに壁に背中をつけて、ずるずると座り込んだ。

（恥ずかしい……でも）

でも、隆弘先輩は引かなかった。むしろ、ゲームっぽくていいって言ってくれた。

もしかして、隆弘先輩も……？

いやいや、それは考えすぎ。でも、希望を持つくらいはいいよね。

◇◇◇◇

机に向かって、今日習ったことを復習する。

```javascript
let heroName = "隆弘先輩";
let lovePoint = 100;
```

さっき打ったコード。消そうと思ったけど、そのまま残してある。

100点。満点の好感度。これが私の、隆弘先輩への今の気持ち。

でも、隆弘先輩の私への好感度は？幼馴染として？後輩として？それとも……。

窓の外を見ると、隆弘先輩の部屋の明かりがついている。きっと、プログラミングをしているんだろう。

明日は、もっと複雑なことを教えてもらえるらしい。配列？for文？よくわからないけど、隆弘先輩が教えてくれるなら、きっと理解できる。

そして、いつか。

いつか、この気持ちを変数じゃなくて、言葉で伝えられる日が来るといいな。

スマホを手に取り、お気に入りの乙女ゲームアプリを開く。攻略キャラの一人が、なんとなく隆弘先輩に似ている気がして選んだゲーム。

「現実もゲームみたいに、選択肢が見えればいいのに」

そうつぶやきながら、私は画面をタップした。

でも、現実はゲームじゃない。だからこそ、ドキドキするし、怖いし、でも楽しい。

隆弘先輩と作るゲーム。それは、私たちの関係も少しずつ変えていくかもしれない。

そんな期待を胸に、私は明日を待つ。

明日も、隆弘くんをドアの前で待とう。一緒に登校して、学校では「先輩」と呼んで、放課後はプログラミングを教えてもらって。

この特別な日々が、ずっと続きますように。

そして、いつか、ゲームの中じゃなくて、現実で、隆弘先輩との特別なエンディングを迎えられますように。

lovePoint = 100。

でも、きっとこれからもっと、もっと増えていく。そんな予感がした。
