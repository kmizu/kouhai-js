# 第2話 プログラミング言語って何？

　四月二十一日、日曜日。午前十時。

　約束の時間ぴったりに、美久がやってきた。今日は薄い水色のブラウスに、デニムのスカート。カジュアルだけど、どこか清楚な雰囲気がある。

「おはよう、隆弘先輩」

「おはよう」

　美久が部屋に入ってくると、すぐに僕のデスクに向かった。昨日使ったホワイトボードには、ASTの図がそのまま残っている。

「昨日の復習、してきたよ」

　美久が鞄から取り出したのは、びっしりと書き込まれたノートだった。

「へえ、真面目だね」

「だって、隆弘先輩との約束だもん」

　そう言って、美久は僕の隣に座った。また近い。昨日より少しだけ、距離が縮まっている気がする。

「それで、今日は何を教えてくれるの？」

「その前に、ちょっと確認させて」

　僕は美久のノートを覗き込んだ。ASTの概念が、美久なりの言葉でまとめられている。「木構造＝家系図みたい」「評価器＝翻訳機」といった具合に。

「理解度高いね。でも、ここがちょっと違うかな」

　僕が指摘すると、美久は真剣な顔でメモを取り始めた。

（こういうところ、昔から変わらないな）

　ふと、小学生の頃を思い出す。

◇◇◇◇

　僕と美久が初めて会ったのは、小学校入学前だった。

　引っ越してきたばかりの僕は、マンションの中庭で一人遊んでいた。そこに現れたのが、当時五歳の美久だった。

「ねえ、何してるの？」

　人見知りの僕は、最初は返事もできなかった。でも美久は気にせず、僕の隣に座り込んだ。

「私、みく。となりに住んでるの」

「……たかひろ」

「たかひろくん？　よろしくね！」

　それから美久は、毎日のように僕を遊びに誘ってくれた。最初は迷惑だと思っていたけど、いつの間にか一緒にいるのが当たり前になっていた。

　小学校でも、美久はいつも僕の世話を焼いてくれた。

「たかひろくん、また本ばっかり読んでる」

「べつにいいだろ」

「だめ！　外で遊ばないと」

　そんな美久に引っ張られて、僕も少しずつ活発になっていった。

　でも、中学に入ってから関係が変わった。

　美久が一年遅れて入学してきた時、学校では「先輩」と呼ぶようになった。最初は違和感があったけど、美久は「学校では先輩後輩だから」と言って聞かなかった。

（今思えば、あの頃から美久は……）

◇◇◇◇

「隆弘先輩？」

　美久の声で、現実に引き戻される。

「あ、ごめん。ちょっと考え事してた」

「もしかして、難しいこと考えてた？」

「いや、そういうわけじゃ……」

　美久がじっと僕を見つめてくる。その瞳に、なんだか見透かされているような気がした。

「じゃあ、本題に入ろうか」

　僕は話題を変えることにした。

「今日は、プログラミング言語とは何か、もう少し深く説明するよ」

「はい！」

　美久が背筋を伸ばした。真剣な表情だ。

「昨日も少し話したけど、プログラミング言語は人間とコンピュータの架け橋なんだ」

　僕は新しいファイルを開いて、説明を始めた。

「コンピュータが理解できるのは、0と1だけ。機械語って呼ばれるやつだね」

「ゼロイチ……デジタルってやつ？」

「そう。でも、人間が0と1だけでプログラムを書くのは、ほぼ不可能」

　僕は簡単な機械語の例を見せた。美久が顔をしかめる。

「うわ、意味不明……」

「だよね。だから、人間にも分かりやすい形で書けるようにしたのがプログラミング言語」

　ホワイトボードに、簡単な図を描く。

```
人間 → プログラミング言語 → 機械語 → コンピュータ
```

「なるほど！　通訳みたいなもの？」

「まさにその通り」

　美久の理解の速さに、また感心する。

「でも、世の中にはたくさんのプログラミング言語があるよね？　なんで？」

「いい質問だね」

　僕は説明を続けた。言語によって得意分野が違うこと。目的に応じて使い分けること。新しい概念を試すために作られることもあること。

「じゃあ、私たちが作る言語は？」

「教育用、かな。シンプルで理解しやすい言語を目指そう」

「教育用……」

　美久が何か考え込んでいる。

「でも、せっかく作るなら、何か特別な機能も欲しいな」

「特別な機能？」

「うん。例えば……」

　美久が身を乗り出してきた。

「日本語で書けるプログラミング言語！」

「日本語？」

「だって、英語ばっかりだと初心者には難しいじゃない？　もしも、たり、ならば、みたいな日本語で書けたら、もっと親しみやすくなると思うの」

　意外にも、それは的を射た提案だった。実際、日本語プログラミング言語は教育用として有効だという研究もある。

（美久、案外センスあるかも）

「実は、そういう言語もあるんだよ。なでしことか」

　美久の目が輝いた。

「本当？　じゃあ、私たちも日本語が使える言語にしよう！」

　その熱意に、僕も感化される。

「と、とりあえず、基本的な機能から作っていこう」

　僕は咳払いをして、話を戻した。

「昨日作った評価器、覚えてる？」

「うん。ASTを受け取って、計算結果を返すやつ」

「今日は、それをもう少し拡張してみよう」

　僕は新しいコードを書き始めた。

```javascript
// 引き算も追加
const ast2 = {
  type: 'BinaryExpression',
  operator: '-',
  left: {
    type: 'NumberLiteral',
    value: 5
  },
  right: {
    type: 'NumberLiteral',
    value: 3
  }
};

console.log(evaluate(ast2)); // 2
```

「わあ、引き算もできた！」

「次は掛け算と割り算も追加してみよう」

　美久と一緒にコードを書いていく。美久は時々質問をしながら、少しずつ理解を深めていく。

「ねえ、隆弘先輩」

「ん？」

「これって、電卓を作ってるみたいだね」

「ある意味、そうかもしれない。でも、普通の電卓との違いは？」

　美久が考え込む。

「えっと……プログラムで動いてる？」

「もっと本質的な違いがあるよ」

　僕はヒントを出した。

「電卓は、ボタンを押したらすぐ計算する。でも、僕たちが作ってるのは？」

「あ！　式を木構造で表現してから計算してる！」

「正解」

　美久が嬉しそうに微笑む。その笑顔を見て、僕も嬉しくなる。

「この『一度構造化してから実行する』っていうのが、プログラミング言語の本質なんだ」

「へー、深いね」

　美久が感心したように頷く。

「隆弘先輩って、こういう話してる時、すごく楽しそう」

「そう？」

「うん。目がキラキラしてる」

　美久の言葉に、僕は少し照れた。

「研究が好きだからね」

「知ってる。小学生の頃から、難しい本ばっかり読んでたもん」

　美久が昔を懐かしむように言う。

「覚えてる？　私が『一緒に遊ぼう』って言っても、『今、大事なところだから』って」

「ああ……悪かったよ」

「ううん。でも、そんな隆弘くんが好きだった」

　さらりと、とんでもないことを言う。

「え？」

「あ、いや、その……友達として！」

　美久が慌てて言い直す。でも、耳まで真っ赤だ。

（これは……どう反応すればいいんだ）

　気まずい沈黙が流れる。

「そ、それより、次の機能を実装しよう」

　僕は強引に話題を変えた。

「今度は、もう少し複雑な式を扱えるようにする」

「複雑な式？」

「例えば、1 + 2 * 3 みたいな」

　美久が首を傾げる。

「それって、答えは？」

「7だよ」

「え？　9じゃないの？」

「掛け算が先だから」

　演算子の優先順位について説明する。美久は「なるほど」と頷きながら、熱心にメモを取る。

　窓の外を見ると、いつの間にか夕方になっていた。

「今日はここまでにしようか」

「え、もう？」

　美久が名残惜しそうな顔をする。

「明日も学校でしょ？」

「そうだけど……」

　美久が立ち上がりながら、ぽつりと呟いた。

「隆弘先輩と一緒にいる時間、あっという間なんだもん」

　その言葉に、僕の心臓がドキリとした。

（美久……）

「じゃあ、また明日の放課後に」

「うん。楽しみにしてる」

　美久が部屋を出ていく。その後ろ姿を見送りながら、僕は考えていた。

　プログラミング言語を作る。その過程で、美久との距離も変わっていく気がする。

　それが良いことなのか、悪いことなのか。

　まだ、分からない。

　でも、一つだけ確かなことがある。

　美久と過ごす時間が、僕にとっても特別なものになりつつあるということを。
