# 第4話 私も実は...

## 10月6日（木）午前7時10分

今朝は、私が隆弘くんの部屋をノックする番だった。

いつもより5分早い。昨日のことがあってから、なんだか落ち着かなくて。

「美久？早いね」

ドアを開けた隆弘くんは、まだ寝起きの顔。髪がくしゃくしゃで、目をこすりながら立っている。

（かわいい……）

「ご、ごめん。早すぎた？」

「いや、大丈夫。ちょっと待って」

隆弘くんが部屋に戻っていく。その後ろ姿を見ながら、私は昨日のことを思い出していた。

ギャルゲーのブックマーク。『夏色の約束』。

実は私、そのゲーム、プレイしたことがある。

いや、正確には、乙女ゲーム版の『夏色の約束 -For Girls-』を。

同じ世界観で、視点を変えて作られた作品。男性向けの本編も気になって、動画サイトで実況を見たりして……。

（まさか隆弘くんも同じゲームをプレイしてたなんて）

共通の話題があることが嬉しい反面、自分の趣味がバレることへの恐怖もある。

でも、昨日の隆弘くんの反応を見る限り、もしかして……。

◇◇◇◇

## 午後12時30分　教室

「美久ちゃん、最近なんか楽しそう」

お弁当を食べながら、クラスメイトの友香が言う。

「そ、そう？」

「うん。なんかキラキラしてる」

キラキラ。そんな風に見えるのかな。

「文化祭の準備が楽しいだけだよ」

「ふーん。嵐山先輩と一緒だもんね」

友香がニヤニヤする。

「それは関係ないから！」

必死に否定するけど、顔が熱い。

「でもさ、プログラミングなんて難しそうなのに、よく頑張るね」

「意外と面白いんだよ。自分で何かを作れるって」

本当は、隆弘くんが教えてくれるからこそ楽しいんだけど。

「へー。じゃあ、今度作ったゲーム見せてね」

「うん、文化祭で展示するから」

そう言いながら、私は午後の授業のことを考えていた。

今日は何を教えてもらえるんだろう。

そして、昨日みたいな気まずい空気にならないといいな……。

◇◇◇◇

## 午後4時30分　プログラミング部室

部室に入ると、隆弘先輩はもうパソコンの前に座っていた。

「あ、美久。今日は早いね」

「はい。楽しみで」

正直に言うと、隆弘先輩の顔を見るのが少し恥ずかしい。でも、プログラミングを学ぶのは本当に楽しみ。

「今日はCSSをやろうと思ってたんだけど……」

隆弘先輩が言いよどむ。

「どうかしましたか？」

「いや、その前に、昨日の話の続きというか……」

ああ、やっぱり触れるんだ。

私の心臓がドキドキし始める。

「実は僕、ギャルゲーが好きなんだ」

隆弘先輩の告白に、私は息を呑んだ。

まさか、はっきり言ってくれるなんて。

「それで、美久も、もしかして……」

隆弘先輩の問いかけに、私は覚悟を決めた。

ここで隠しても、もう意味がない。

それに、隆弘先輩が正直に話してくれたんだから。

「私も……実は、乙女ゲームが好きです」

言った。言っちゃった。

でも、不思議と後悔はない。むしろ、肩の荷が下りたような感覚。

隆弘先輩の顔に、安堵の表情が広がる。

「そうだったんだ……」

「『夏色の約束』も、For Girls版ですけど、プレイしました」

「本当に！？」

隆弘先輩の目が輝く。

「はい。すごく良いシナリオでしたよね。特に、幼馴染ルートが」

「分かる！幼馴染の葛藤とか、すごくリアルで」

急に会話が弾み始める。

ゲームの話になると、隆弘先輩の目がキラキラして、いつもよりも饒舌になる。

（この人、本当にゲームが好きなんだな）

そして、私も同じ。

今まで誰にも話せなかった趣味の話を、思う存分できる相手がいる。

しかも、それが憧れの隆弘先輩だなんて。

「美久は、どんなゲームが好きなの？」

「学園ものが多いですね。現実的な設定の方が感情移入しやすくて」

「僕も同じだ。ファンタジーより、日常系の方が好き」

共通点がどんどん見つかる。

好きなシナリオライター、音楽、声優さん。

話は尽きない。

「あ、でも」

ふと我に返る隆弘先輩。

「プログラミングの勉強もしないと」

「そ、そうですね」

でも、お互いの顔を見て、くすっと笑ってしまう。

「でも、これでゲーム制作の話もしやすくなったね」

隆弘先輩の言葉に、私は頷く。

「はい。遠慮なく、理想のゲームを作れます」

「じゃあ、CSSで画面を可愛くする方法を教えるよ」

隆弘先輩がコードを打ち始める。

```css
#text-box {
    background-color: rgba(255, 255, 255, 0.9);
    border: 2px solid #ff69b4;
    border-radius: 10px;
    padding: 20px;
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}
```

「これで、テキストボックスに可愛い枠がつく」

「わあ、ピンクの枠！」

「美久が好きそうかなって」

隆弘先輩の心遣いが嬉しい。

「あの、先輩」

「ん？」

「これからは、遠慮なくゲームの話もしていいですか？」

私の問いかけに、隆弘先輩は優しく微笑んだ。

「もちろん。むしろ、そうしてくれると嬉しい」

この瞬間、私たちの関係が一歩前進した気がした。

◇◇◇◇

## 午後6時30分　プログラミング部室

CSSの基礎を一通り学んで、画面がどんどん可愛くなっていく。

「背景画像も設定できるんですね」

「うん。ゲームの雰囲気に合わせて変えられる」

隆弘先輩が、桜の画像を背景に設定する。

春の学園を舞台にしたゲーム。

「美久はどんなストーリーにしたい？」

「幼馴染との恋愛ものがいいです」

思わず本音が出てしまった。

隆弘先輩の手が一瞬止まる。

「幼馴染、か……」

なんだか意味深な反応。

「先輩は？」

「僕も、幼馴染ものは好きだよ。距離感が難しいけど、それがまた良いんだよね」

隆弘先輩の言葉に、胸がきゅっとなる。

私たちも、幼馴染。

この関係から一歩踏み出すのは、確かに難しい。

でも、ゲーム制作を通じて、少しずつ距離が縮まっている気がする。

「じゃあ、主人公とヒロインは幼馴染設定で」

「はい！」

二人で設定を詰めていく。

気がつけば、もう外は暗くなっていた。

◇◇◇◇

## 午後7時45分　自室

部屋に戻って、今日のことを振り返る。

ついに、お互いの秘密を打ち明けた。

隆弘先輩はギャルゲーマー。

私は乙女ゲーマー。

なんだか、運命的な気がする。

スマホを開いて、お気に入りの乙女ゲームアプリを見る。

でも、今はプレイする気分じゃない。

だって、現実の方がずっとドキドキするから。

明日は金曜日。

週末も一緒にゲーム制作ができるかな。

今度は、お互いのおすすめゲームを教え合うのもいいかも。

ベッドに寝転がって、天井を見上げる。

隆弘先輩と過ごす時間が、どんどん特別なものになっていく。

プログラミング。

ゲーム。

そして、これから生まれる物語。

全部が、私たちだけの大切な思い出になる。

「隆弘先輩……」

小さくつぶやいて、顔を枕に埋める。

明日が待ち遠しくて、きっと今夜は眠れない。

でも、それも悪くない。

だって、こんなに幸せな気持ちで眠れないなんて、初めてだから。

窓の外を見ると、隆弘先輩の部屋の明かりがまだついている。

もしかして、同じ気持ちでいてくれたら……。

そんな淡い期待を胸に、私は目を閉じた。

明日も、素敵な一日になりますように。
