# 第8話 本格始動、私たちのゲーム

## 10月11日（火）午前7時20分

「隆弘くん、今日から本格的にゲーム作るんだよね？」

エレベーターの中で、美久が期待に満ちた声で聞いてくる。

「ああ。基礎は一通り教えたから、いよいよ実装だ」

「楽しみ！どんなゲームにする？」

美久の瞳がキラキラしている。

「それは、今日みんなで決めよう」

◇◇◇◇

## 午後4時30分　プログラミング部室

今日は特別な日だ。プログラミング部と文芸部の合同会議。

部室には、両部の主要メンバーが集まっていた。

「それでは、ゲームの詳細を決めていきましょう」

文芸部部長の佐藤先輩が議事を進行する。

「まず、ストーリーの概要から」

美久が立ち上がり、みんなの前で説明を始めた。

「舞台は、私たちと同じような高校。主人公は、幼馴染への恋心を自覚していく物語です」

（幼馴染……）

なんだか、リアルすぎる設定だ。

「主人公の性別は？」

プログラミング部の田中が質問する。

「プレイヤーが選べるようにしたいです」

美久の提案に、みんなが頷く。

「技術的には可能だよね、嵐山？」

「ああ。変数で管理すれば簡単だ」

僕は自信を持って答えた。

「それと、攻略キャラクターは3人用意します」

美久がキャラクター設定を説明し始める。

1. 優しい先輩キャラ（プログラミングが得意）
2. 元気な同級生キャラ（スポーツ万能）
3. クールな後輩キャラ（芸術系）

「なんか、1番のキャラ、誰かに似てない？」

田中がニヤニヤしながら言う。

「き、気のせいです！」

美久が真っ赤になって否定する。

（僕がモデル……？）

◇◇◇◇

## 午後5時30分　役割分担

「じゃあ、役割を決めよう」

僕が提案すると、みんなが注目する。

「プログラミング：嵐山、田中、プログラミング部員」
「シナリオ：美久、佐藤先輩、文芸部員」
「イラスト：美久、文芸部の絵が得意な人」
「音楽：フリー素材を使用」

「美久、イラストも描けるの？」

「少しだけ……立ち絵くらいなら」

謙遜する美久だが、きっと上手いんだろう。

「BGMは、著作権フリーのサイトから選ぼう」

「『魔王魂』とか有名ですよね」

ゲーマーらしい知識が飛び交う。

◇◇◇◇

## 午後6時　プログラミング開始

みんなが帰った後、美久と二人で残った。

「さて、まずは基本システムから作ろう」

僕はVSCodeを開き、新しいプロジェクトを作成した。

```javascript
// ゲームの基本設定
const gameConfig = {
    title: "きみと紡ぐ物語",
    version: "1.0.0",
    author: "京都中央学園 プログラミング部×文芸部"
};

// プレイヤー情報
let player = {
    name: "",
    gender: "",
    currentScene: 0
};

// キャラクターの好感度
let affection = {
    senpai: 0,
    classmate: 0,
    kouhai: 0
};
```

「わあ、本格的！」

美久が画面を見つめる。

「これが、僕たちのゲームの土台だ」

「『きみと紡ぐ物語』……素敵なタイトル」

美久が考えたタイトルだ。

「次は、シーン管理システムを作る」

```javascript
// シーンデータ
const scenes = [
    {
        id: 0,
        background: "classroom.jpg",
        character: "none",
        text: "4月、新学期。私の物語が始まる――",
        choices: [
            { text: "教室に入る", next: 1 },
            { text: "少し待つ", next: 2 }
        ]
    }
];

// シーン表示関数
function displayScene(sceneId) {
    const scene = scenes[sceneId];
    // 背景画像を変更
    document.getElementById('background').src = scene.background;
    // テキストを表示
    document.getElementById('text').innerText = scene.text;
    // 選択肢を表示
    displayChoices(scene.choices);
}
```

「なるほど！シーンごとにデータを管理するんですね」

美久が理解の早さを見せる。

「そう。これで物語の流れを制御できる」

◇◇◇◇

## 午後7時30分　初めての動作確認

「じゃあ、動かしてみよう」

緊張の一瞬。ブラウザでindex.htmlを開く。

画面に、ゲームのタイトルが表示された。

『きみと紡ぐ物語』

スタートボタンをクリックすると、最初のシーンが表示される。

「動いた！」

美久が歓声を上げる。

まだテキストと選択肢だけの簡素な画面だけど、確かに動いている。

「すごい……私たちのゲームだ」

美久の目に、涙が浮かんでいる。

「まだ始まったばかりだよ」

「でも、嬉しい。隆弘先輩と一緒に作れて」

その言葉に、僕の胸が熱くなる。

「俺も、美久と作れて嬉しい」

素直な気持ちを口にすると、美久の頬が赤くなった。

◇◇◇◇

## 午後8時　帰り道

「今日から、本当のゲーム制作が始まったね」

暗い道を歩きながら、美久が言う。

「ああ。でも、まだまだやることは山積みだ」

「イラストも頑張って描かなきゃ」

「無理しなくていいよ」

「ううん、頑張る。だって……」

美久が立ち止まる。

「隆弘先輩が頑張ってプログラミングしてくれるから、私もできることを精一杯やりたい」

その真剣な眼差しに、胸を打たれる。

「一緒に、最高のゲームを作ろう」

「はい！」

美久の笑顔が、街灯に照らされて輝く。

◇◇◇◇

## 午後9時　自室

部屋に戻って、今日書いたコードを見返す。

まだ基本的な部分だけだけど、確実に形になり始めている。

『きみと紡ぐ物語』

このタイトルが、僕と美久の関係を表しているような気がした。

プログラミングを通じて、ゲーム制作を通じて、僕たちは確実に新しい物語を紡いでいる。

スマホが震えた。美久からのLINEだ。

『今日はありがとうございました！明日はキャラクターの立ち絵を持っていきます』

『楽しみにしてる』

『あの、優しい先輩キャラ、隆弘先輩をイメージして描いてもいいですか？』

ドキッとする質問。

『好きにしていいよ』

『やった！じゃあ、かっこよく描きますね』

美久のメッセージを見て、顔がにやけてしまう。

明日が楽しみだ。

美久の描いたイラストがゲームに実装される日。

そして、二人で作り上げるゲームが完成する日。

その日まで、全力で頑張ろう。

窓の外を見ると、隣の部屋の明かりがついている。

きっと美久も、イラストを描いているんだろう。

（一緒に頑張ろう、美久）

心の中でつぶやいて、僕も作業を再開した。

僕たちの物語は、まだ始まったばかりだ。
