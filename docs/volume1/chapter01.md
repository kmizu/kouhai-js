# 第1話 幼馴染の新しい一歩

## 10月3日（月）午前7時15分

「隆弘くん、遅いよー！」

マンションの廊下に響く声で、僕は慌てて玄関のドアを開けた。

そこには、いつものように腕を組んで待っている幼馴染の姿があった。河内美久。隣の701号室に住む、小学校からの腐れ縁だ。

「ごめん、昨日遅くまでコード書いてて」

「またプログラミング？本当に好きだよね」

美久はあきれたような笑顔を浮かべながら、僕の隣に並んだ。高校の制服に身を包んだ彼女は、去年の中学生の頃より少し大人っぽく見える。

（いや、単に高校の制服効果か？）

エレベーターを待つ間、美久は何か言いたそうに僕の顔を見上げていた。身長差は約20センチ。小学生の頃は同じくらいだったのに、いつの間にか僕の方が大きくなってしまった。

「どうかした？」

「あのね、学校で相談があるんだけど」

「学校で？」

エレベーターが到着し、二人で乗り込む。7階から1階まで、約30秒の密室。美久の髪から、ほのかにシャンプーの香りが漂ってきて、なぜか意識してしまう。

「うん。ちゃんと先輩として、相談に乗ってもらいたいから」

先輩。その言葉に、僕は苦笑した。

家では「隆弘くん」と呼ぶくせに、学校では律儀に「先輩」と呼ぶ美久。その使い分けが、なんだか面白い。

◇◇◇◇

京都中央学園への通学路は、いつもの三条会商店街を抜けていく。朝の商店街は、開店準備で忙しそうだ。

「それで、相談って何？今聞いちゃダメなの？」

歩きながら僕が聞くと、美久は少し頬を染めた。

「だって、ここで話すと、学校に着く前に話が終わっちゃうもん」

「そんなに長い相談なの？」

「ううん、そうじゃなくて……」

美久は言いよどみ、前を向いたまま小さくつぶやいた。

「先輩として、ちゃんと聞いて欲しいから」

その言葉の意味が分からないまま、僕たちは学校への坂道を上り始めた。

校門をくぐると、美久の態度が微妙に変わる。少し距離を取り、背筋を伸ばす。

「それじゃ、また放課後に」

「ああ、放課後ね」

美久は軽く会釈をして、1年生の教室がある棟へ向かっていった。

（相談か……何だろう）

僕は首を傾げながら、2年生の教室へ向かった。

◇◇◇◇

## 午後4時30分　プログラミング部室

放課後、いつものように部室でコードを書いていると、控えめなノックの音が聞こえてきた。

「失礼します」

ドアを開けて入ってきたのは、予想通り美久だった。でも、朝とは違って、なんだか緊張した面持ちをしている。

「あ、隆弘先輩」

学校での呼び方。でも、部室には僕しかいない。

「他に誰もいないよ」

僕がそう言うと、美久は少しほっとしたような顔をした。

「よかった。実は、先輩にお願いがあって」

美久は僕の隣の椅子に座り、真剣な表情で切り出した。

「文化祭で、文芸部として何か新しいことに挑戦したいんです」

「新しいこと？」

「はい。それで……ノベルゲームを作りたいんです」

ノベルゲーム。その言葉に、僕の心臓が跳ねた。

（まさか美久がノベルゲームなんて言葉を……）

僕の動揺に気づかず、美久は続けた。

「でも、文芸部には誰もプログラミングができる人がいなくて。それで、隆弘先輩に……その、JavaScriptを教えていただけませんか？」

美久は両手を膝の上で組み、上目遣いで僕を見つめた。学校では後輩として振る舞う彼女が、今は幼馴染の顔も見せている。

「もちろん、部活の合同企画として正式に申請します。でも、その前に、隆弘先輩の気持ちを聞きたくて」

僕は美久の真剣な眼差しを受け止めた。プログラミングを教える。それも、ノベルゲーム制作のために。

（これは……願ってもないチャンスかもしれない）

「もちろん、いいよ」

僕の返事に、美久の顔がぱっと明るくなった。

「本当ですか？」

「ああ。むしろ、プログラミング部としても文化祭の展示内容を考えていたところだから、ちょうどいい」

「よかった！私、隆弘くん……じゃなくて、先輩となら、きっと素敵なゲームが作れると思ってたんです」

言い直す美久が可愛くて、僕は思わず笑ってしまった。

「ここには僕しかいないんだから、隆弘くんでいいよ」

「でも、けじめは大事です」

真面目な顔で言う美久。でも、その頬は少し赤い。

「じゃあ、さっそく基礎から始めようか」

僕はパソコンの画面をつけ、エディタを開いた。

「まず、ノベルゲームを作るには、HTMLとJavaScriptの基礎を理解する必要があるんだ」

「えいちてぃーえむえる？」

「そう。Webページの構造を作る言語だよ」

僕は簡単なHTMLのコードを打ち始めた。

```html
<!DOCTYPE html>
<html>
<head>
    <title>私たちのゲーム</title>
</head>
<body>
    <h1>ようこそ</h1>
    <p>ここから物語が始まります。</p>
</body>
</html>
```

「わあ、意外とシンプルなんですね」

美久は画面を食い入るように見つめている。その横顔が真剣で、いつもの幼馴染とは違う一面を見た気がした。

「この`<h1>`が見出しで、`<p>`が段落。こうやってタグで囲むことで、文章に意味を持たせるんだ」

「なるほど……先輩、本当に詳しいんですね」

尊敬の眼差しを向けられて、なんだか照れくさい。

「プログラミング部の部長だからね」

「でも、小学生の頃から、隆弘くんはパソコンが好きだったもんね」

急に昔の呼び方に戻る美久。

「あ、すみません。つい……」

「いいよ、その方が自然だし」

僕たちは顔を見合わせて、小さく笑った。

◇◇◇◇

「じゃあ、美久も打ってみる？」

「え、もうですか？」

「習うより慣れろ、だよ」

僕は席を立ち、美久に場所を譲った。美久は恐る恐るキーボードに手を伸ばす。

「えっと、最初は……」

一文字ずつ、慎重にタイピングしていく美久。小学生の頃、一緒に宿題をした時のことを思い出す。あの頃も、真剣な顔で鉛筆を動かしていた。

（変わらないな、この真剣さは）

「先輩、これで合ってますか？」

「うん、完璧だよ」

「よかった！」

嬉しそうに微笑む美久。その笑顔を見ていると、なぜか胸が温かくなる。

「プログラミングって、思ったより楽しいかも」

「そう言ってもらえると嬉しいな」

「だって、何もないところから何かを作れるんですよね？それってすごくクリエイティブ」

美久の言葉に、僕の心臓がまた跳ねた。

プログラミングの楽しさを、美久が理解してくれている。それは、予想以上に嬉しいことだった。

◇◇◇◇

## 午後6時　マンションへの帰り道

「今日はありがとうございました、先輩」

夕焼けに染まる通学路を、二人で歩く。学校を出てもまだ「先輩」と呼ぶ美久に、僕は苦笑した。

「もう学校じゃないんだから」

「でも、今日は先輩として教えてもらったから」

美久は少し照れたように微笑む。

「それに、隆弘先輩って呼ぶのも、なんだか新鮮で」

「新鮮？」

「だって、いつもは隆弘くんだもん。でも、学校で先輩として頼れる存在だって改めて実感できて、なんだか……」

美久は言葉を濁し、頬を赤らめた。

「なんだか？」

「……かっこいいなって」

小さな声でつぶやかれた言葉に、今度は僕が赤面する番だった。

「な、何言ってるんだよ」

「本当のことだもん」

むくれる美久。その姿が可愛くて、僕はますます動揺してしまう。

商店街の明かりが灯り始める中、僕たちは並んで歩き続けた。いつもと同じ帰り道なのに、なぜか特別な時間に感じられた。

マンションのエントランスに着くと、美久は振り返った。

「明日も、教えてくれる？」

「もちろん」

「じゃあ、明日は変数について教えてください」

「え、もう予習したの？」

「ちょっとだけ。だって、早くゲームを作れるようになりたいから」

意欲的な美久に、僕は感心した。

「分かった。明日は変数とif文あたりをやろう」

「はい！楽しみにしてます」

エレベーターに乗り、7階に到着。それぞれの部屋の前で立ち止まる。

「おやすみ、隆弘くん」

家の前では、やっぱり昔からの呼び方に戻る美久。

「おやすみ、美久」

ドアが閉まる音を聞きながら、僕は自分の部屋に入った。

◇◇◇◇

机に向かい、パソコンを起動する。明日の準備をしようと思ったのに、なぜか集中できない。

頭の中に浮かぶのは、真剣にキーボードを打つ美久の姿。「かっこいい」と言われた時の、あの表情。

（これは、まずいかもしれない）

幼馴染で、後輩で。そんな美久と一緒にゲームを作る。

プログラミングを通じて、新しい関係が始まりそうな予感がした。

画面には、今日作ったHTMLファイルがまだ開いている。

『私たちのゲーム』

そのタイトルを見つめながら、僕は明日のことを考えた。

変数の説明は、ゲームの好感度システムに例えると分かりやすいかもしれない。でも、それを説明する時、美久はどんな反応をするだろうか。

（まさか、美久もゲームとか……いや、ないか）

窓の外を見ると、隣の部屋の明かりが灯っていた。美久も今頃、今日習ったことを復習しているのかもしれない。

そう思うと、なんだか嬉しくなって、僕は明日の準備に取り掛かった。

美久と作る『私たちのゲーム』。

それは、先輩後輩という関係に、新しい色を加えてくれそうな気がした。

ふと、本棚に目をやる。そこには、こっそり集めているゲームのパッケージが並んでいる。

（いつか、美久にも話せる日が来るかな）

そんな淡い期待を胸に、僕はコードを書き始めた。

明日もまた、美久が部室にやってくる。

「先輩」と呼ばれながら、でも時々「隆弘くん」になる、あの不思議な距離感。

それが、これからどう変わっていくのか。

期待と不安を胸に、僕は夜更けまでキーボードを叩き続けた。
