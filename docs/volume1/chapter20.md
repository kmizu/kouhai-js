# 第20話 Promise

　文化祭から一週間後の日曜日。

　今日は、ヒロ先輩と初めてのデート。場所は、京都市動物園。

「パンダ見たい！」

「いいよ」

　手をつないで園内を回る。まだ慣れなくて、ドキドキする。

「美久の手、ちっちゃいね」

「先輩の手が大きいんです」

　そんな他愛もない会話が、幸せ。

　ベンチに座って、持ってきたお弁当を食べる。

「美味しい」

「本当ですか？」

「うん。美久、料理も上手なんだね」

　褒められて嬉しい。昨日の夜、頑張って作った甲斐があった。

「そういえば」

　ヒロ先輩が切り出す。

「インタラクティブ小説の続編、進んでる？」

「はい！恋人編のプロット、もうできてます」

「見せて」

　スマホでメモを見せる。今度は、恋人になってからの日常を描く予定。

「いいね。また手伝うよ」

「ありがとうございます」

　でも、今度は本当の意味で一緒に作れる。恋人として。

　動物園を出て、鴨川沿いを歩く。秋の風が心地いい。

「先輩」

「なに？」

「Promise、覚えてますか？」

　突然の質問に、ヒロ先輩が少し考える。

「非同期処理の？」

「はい。約束は必ず守られるって」

　立ち止まって、ヒロ先輩を見上げる。

「私、先輩と約束したいことがあります」

「どんな？」

「これからも、ずっと一緒にいてください」

　真剣に言う。ヒロ先輩も真剣な顔で答える。

「Promise」

　プログラミング用語だけど、今は本当の約束。

```javascript
const promise = new Promise((resolve) => {
    resolve('ずっと一緒にいる');
});

promise.then(result => {
    console.log(result); // 永遠に
});
```

　頭の中で、そんなコードを思い浮かべる。

「あ、でも」

　ヒロ先輩が付け加える。

「エラー処理も必要かな」

「エラーなんて起きません！」

　むっとして言うと、ヒロ先輩が笑う。

「そうだね。絶対にresolveする」

　鴨川の水面がキラキラ光っている。カップルや家族連れが、楽しそうに過ごしている。

　私たちも、その中の一組。

「美久」

「はい？」

「好きだよ」

　突然の告白に、顔が熱くなる。

「私も、大好きです」

　ヒロ先輩が優しく頭を撫でてくれる。幸せ。

　家に帰る道、いつもの三条会商店街を通る。

「あ、みやこでコーヒー飲んでいきませんか？」

「いいね」

　二人でよく勉強した喫茶店。でも、今日は恋人として。

　窓際の席に座って、コーヒーを飲む。

「懐かしいね、ここで最初にJavaScript教えたの」

「document.writeから始まりましたね」

「よく覚えてる」

「忘れるわけないです」

　大切な思い出。全部覚えている。

　ノートを取り出して、新しいページを開く。

「何書くの？」

「今日のこと」

　日付を書いて、今日の出来事を記録し始める。

『初めてのデート。動物園。手をつないだ。お弁当。Promise。』

　箇条書きだけど、これで十分。記憶は心に刻まれている。

「僕も書いていい？」

「どうぞ」

　ヒロ先輩がペンを取って、続きを書く。

『美久と過ごす日曜日。これからも、ずっと。』

　そして最後に、

『Promise.resolve(true);』

　と書き加えた。

　二人で顔を見合わせて、笑う。

　マンションに着いて、エレベーターに乗る。

「今日は楽しかった」

「私も」

　6階に着いて、降りる前に振り返る。

「先輩」

「ん？」

「また明日」

「うん、また明日」

　いつもの挨拶。でも、意味が違う。恋人としての「また明日」。

　部屋に戻って、今日撮った写真を見返す。動物園での二人の写真。幸せそうな笑顔。

　パソコンを開いて、インタラクティブ小説の続編を書き始める。

　主人公とプログラマーの先輩の、甘い日常。それは、私たちの物語でもある。

　JavaScriptを教えてもらうことから始まった関係。今では、人生のパートナー。

　コードを書きながら思う。人生も、プログラミングみたい。思い通りにいかないこともある。エラーも出る。でも、デバッグして、より良いものにしていける。

　ヒロ先輩と一緒なら、どんなエラーも乗り越えられる。

　窓の外を見ると、7階の部屋に明かりが灯っている。ヒロ先輩も起きてる。

　LINEを送る。

『おやすみなさい』

　すぐに返事が来た。

『おやすみ。いい夢を』

　そして、もう一つ。

『愛してる』

　短いメッセージ。でも、心がいっぱいになる。

『私も愛してます』

　返信して、スマホを置く。

　ベッドに入って、今日一日を振り返る。

　Promise。約束。ヒロ先輩との約束は、必ず守られる。エラーなんて起きない。

　だって、これは愛のコードだから。

　幸せな気持ちで、ゆっくりと眠りについた。

（完）
