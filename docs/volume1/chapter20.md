# 第20話 MikuLangの完成と告白

　五月九日、木曜日。

　放課後の屋上。今日は特別な場所を選んだ。MikuLangの完成を祝うため、そして——

「わあ、夕日がきれい」

　美久が手すりにもたれて、オレンジ色に染まる空を見上げる。

「今日でMikuLang、完成だね」

「うん。最後の仕上げをしよう」

　持参したノートパソコンを開く。屋上のベンチに二人で座る。

◇◇◇◇

「完成版のMikuLangを動かしてみよう」

```javascript
// MikuLang完成版のデモプログラム
const completionDemo = `
  「MikuLang 完成おめでとう！」を表示する
  
  作者リストを["隆弘", "美久"]に設定する
  
  作者リストの各要素を名前として｛
    名前と「さん、お疲れ様でした」を表示する
  ｝
  
  開発日数を10に設定する
  思い出の数を無限大に設定する
  
  感謝のメッセージを定義する（）｛
    「このプログラミング言語は」を表示する
    「二人の協力なしには完成しませんでした」を表示する
    「本当にありがとう」を表示する
  ｝
  
  感謝のメッセージ（）
`;

// 実行
const tokenizer = new Tokenizer(completionDemo);
const tokens = tokenizer.tokenize();
const parser = new JapaneseParser(tokens);
const ast = parser.parseProgram();
const evaluator = new MikuLangEvaluator();
evaluator.run(ast);
```

「動いた！」

　美久が喜びの声を上げる。

「私たちの言語が、本当に動いてる」

◇◇◇◇

「MikuLangの特徴をまとめてみよう」

　ノートパソコンにリストを作る。

```
MikuLang 1.0 の特徴：
- 完全な日本語プログラミング
- 自然な日本語構文
- 初心者に優しいエラーメッセージ
- JavaScriptで実装された処理系
- 変数、関数、制御構造を完備
- 拡張可能な設計
```

「すごい。本物のプログラミング言語だ」

　美久の目が輝いている。

「美久がいたから完成できた」

「隆弘先輩こそ」

◇◇◇◇

　夕日が水平線に近づいてきた。今しかない。

「美久」

「なに？」

　深呼吸をして、準備していたプログラムを見せる。

```javascript
// 隆弘の告白プログラム
const takahiroConfession = `
  美久への想いを定義する（）｛
    初めて会った日を"幼稚園の入園式"に設定する
    
    もし 時間が経つほど｛
      想いの強さを増やし続ける
    ｝
    
    今の気持ちを"愛してる"に設定する
    
    「美久、ずっと前から好きでした」を表示する
    「プログラミングを通じて、もっと好きになりました」を表示する
    「これからも一緒にいてください」を表示する
    
    返事を待つ
  ｝
  
  美久への想い（）

```

◇◇◇◇

　美久が画面を見つめる。頬が赤く染まっていく。

「これ、MikuLangで書いた告白？」

「うん。実行していい？」

　美久が小さく頷く。

　プログラムを実行する。画面に文字が表示される。

```
美久、ずっと前から好きでした
プログラミングを通じて、もっと好きになりました
これからも一緒にいてください
```

◇◇◇◇

　美久の目に涙が浮かぶ。

「ずるい。こんな告白」

「ごめん。でも、これが僕の本当の気持ち」

　美久もノートパソコンに手を伸ばす。

「私も書く」

```javascript
// 美久の返事プログラム
const mikuReply = `
  隆弘先輩への返事を定義する（）｛
    私の気持ちを"大好き"に設定する
    
    幼い頃からを思い出として｛
      「ずっと隆弘先輩を見てました」を表示する
      「プログラミングを教わって」を表示する
      「もっともっと好きになりました」を表示する
    ｝

```

◇◇◇◇

　プログラムを実行すると、美久の気持ちが画面に表示される。

```
ずっと隆弘先輩を見てました
プログラミングを教わって
もっともっと好きになりました
私も愛してます
ずっと一緒にいさせてください
```

　そして、美久が立ち上がって、僕に抱きついてきた。

「ハグを実行する、だから」

　涙声で言う美久を、しっかりと抱きしめる。

◇◇◇◇

「美久」

「隆弘先輩」

　夕日を背景に、見つめ合う。

「付き合ってください」

「はい」

　即答だった。

　そっと顔を近づける。美久も目を閉じる。

　夕日に照らされながら、キスをした。

　初めてのキス。甘くて、温かくて、幸せ。

◇◇◇◇

「MikuLangの最初のプログラムが、告白になっちゃった」

　美久が照れながら言う。

「記念すべき第一号プログラム」

「ずっと保存しておく」

　二人で笑い合う。

　手を繋いで、夕日を見る。

「隆弘先輩」

「ん？」

「MikuLangで、もっといろんなもの作ろうね」

「うん。二人で」

◇◇◇◇

　屋上から降りる時、美久が言った。

「今日は特別な日になったね」

「MikuLangの誕生日で、僕たちの記念日」

「5月9日。忘れない」

　階段を降りながら、これからのことを話す。

　MikuLangの改良、新しいプログラムの作成、そして二人の未来。

◇◇◇◇

　校門を出たところで、美久が立ち止まった。

「あのプログラム、本当に嬉しかった」

「美久のプログラムも」

「プログラミングって、気持ちも伝えられるんだね」

　その通りだと思う。

　コードに込めた想いは、確実に相手に届く。

◇◇◇◇

　家に帰ってから、二人でMikuLangのリポジトリを作った。

```
MikuLang - A Japanese Programming Language
Created with love by Takahiro & Miku
First program: Love confession
Status: Complete and evolving
```

「これから、たくさんの人に使ってもらえるといいね」

　美久の夢が広がる。

「きっと使ってもらえる。初心者に優しい言語だから」

◇◇◇◇

　その夜、美久からメッセージが来た。

『今日はありがとう。MikuLangも、告白も、全部嬉しかった』

『こちらこそ。美久がいなければ、どちらも実現しなかった』

『明日から、恋人として、開発パートナーとして、よろしくお願いします』

『こちらこそ、よろしく』

　そして、最後のメッセージ。

『愛してる』

『僕も愛してる』

　MikuLangの完成と、恋の成就。

　プログラミングが繋いだ、二つの心。

　明日からも、二人で新しいコードを書いていこう。

　愛という名の、永遠のプログラムを。
