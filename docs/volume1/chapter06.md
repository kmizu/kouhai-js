# 第6話 変数と好感度システム

## 10月8日（土）午前10時

「美久、今日は大丈夫？」

土曜日の朝、僕は美久の部屋のインターホンを鳴らした。

「あ、隆弘くん！もちろん！」

ドアが開くと、私服姿の美久が立っていた。白いブラウスにベージュのカーディガン、チェック柄のスカート。

（かわいい……）

「どうしたの？じろじろ見て」

「い、いや、私服も似合うなって」

美久の頬が赤くなる。

「ありがとう。隆弘くんも、私服だとちょっと大人っぽいね」

今度は僕が照れる番だった。

「それで、今日は僕の部屋でやろうと思うんだけど」

「え、隆弘くんの部屋？」

美久が驚いたような顔をする。

「プログラミング部の部室、土日は使えないから。それに、家の方がリラックスできるし」

「そ、そうだね。お邪魔します」

◇◇◇◇

## 午前10時15分　隆弘の部屋

「わあ、意外と片付いてる」

部屋に入った美久が、きょろきょろと見回す。

「一応、掃除したから」

本当は朝から必死で片付けたけど、そんなこと言えない。

「あ、本棚！」

美久が目を輝かせる。そこには、プログラミングの参考書に混じって、ゲームの攻略本やラノベが並んでいる。

「『星降る夜のラプソディ』の設定資料集！」

「あ、それは……」

慌てて隠そうとするが、もう遅い。

「いいじゃないですか。私も持ってますよ、For Girls版の」

美久がにっこり笑う。

「そ、そうか」

なんだか、お互いの部屋を見せ合うのも悪くない気がしてきた。

「じゃあ、始めようか」

僕はノートPCを開き、美久を隣に座らせた。

近い。いつもの部室より距離が近い。美久のシャンプーの香りがふわっと漂ってくる。

（集中、集中）

「今日は、変数についてもっと詳しくやろう」

◇◇◇◇

## 午前11時　プログラミングレッスン

「変数は、データを入れる箱みたいなもの」

僕はホワイトボードがわりのノートに図を描きながら説明する。

「例えば、ゲームの好感度システム」

```javascript
let mikuAffection = 0;  // 美久の好感度
let takahiroAffection = 0;  // 隆弘の好感度
```

「あ、私の名前！」

美久が嬉しそうに言う。

「例だから……でも、実際のゲームでもこんな感じで管理するんだ」

```javascript
// 選択肢によって好感度が変化
function makeChoice(choice) {
    if (choice === "一緒に帰る") {
        mikuAffection += 10;
        console.log("美久の好感度が上がった！");
    } else if (choice === "用事がある") {
        mikuAffection -= 5;
        console.log("美久の好感度が下がった...");
    }
}
```

「なるほど！選択によって数値が変わるんですね」

美久が真剣にメモを取る。

「そう。で、この好感度によってエンディングが変わったり」

```javascript
function checkEnding() {
    if (mikuAffection >= 80) {
        console.log("ハッピーエンド！");
    } else if (mikuAffection >= 50) {
        console.log("ノーマルエンド");
    } else {
        console.log("バッドエンド...");
    }
}
```

「80以上でハッピーエンド……」

美久がぽつりとつぶやく。

「現実にも好感度が見えたらいいのにね」

冗談のつもりで言った言葉に、美久が真剣な顔で答える。

「見えない方がいいこともありますよ」

「どうして？」

「だって、数値じゃ測れない気持ちもあるから」

美久の言葉に、僕はドキッとした。

彼女は一体、誰のことを想って言っているんだろう。

◇◇◇◇

## 午後12時30分　昼食休憩

「お昼、どうする？」

「あ、私がお弁当作ってきました！」

美久が持参した紙袋から、タッパーを取り出す。

「え、僕の分も？」

「だって、一緒に食べた方が美味しいでしょ？」

当たり前のように言う美久。

「それに、いつも教えてもらってるお礼です」

タッパーの中身は、唐揚げ、卵焼き、ほうれん草のおひたし。どれも美味しそうだ。

「すごい、本格的」

「えへへ、実は料理も趣味なんです」

意外な一面を知って、また美久のことが好きになる。

いや、好きって……。

「どうしたの？固まって」

「い、いや、美味しそうだなって」

「召し上がれ！」

一緒に食べるお弁当は、本当に美味しかった。

◇◇◇◇

## 午後2時　プログラミング再開

「さて、次はif文を使って実際に選択肢を作ってみよう」

午後の部が始まった。お腹もいっぱいで、少し眠たそうな美久。

「大丈夫？眠い？」

「大丈夫です！頑張ります」

でも、まぶたが重そうだ。

「少し休憩する？」

「でも……」

「15分だけ」

僕がそう言うと、美久は安心したように目を閉じた。

そして、そのまま僕の肩にもたれかかってきた。

「み、美久？」

返事はない。寝てしまったらしい。

規則正しい寝息が聞こえる。

（どうしよう……）

動けない。動きたくない。

美久の重さと温もりを感じながら、僕は画面を見つめる。

```javascript
let currentAffection = 100;  // 今の好感度
```

100点満点。

もし、現実に好感度があるなら、今の僕の美久への好感度は……。

「ん……」

美久が小さく声を出して、さらに僕に寄りかかってくる。

この状況、if文で表すなら。

```javascript
if (美久が寝ている && 距離が近い) {
    心拍数 = 最大値;
    理性 = 限界;
}
```

なんて、プログラマーらしくない思考だ。

◇◇◇◇

## 午後4時　目覚め

「はっ！」

美久が急に目を覚ました。

「ご、ごめんなさい！私、寝ちゃって……」

顔を真っ赤にして謝る美久。

「いいよ。疲れてたんだろ」

「でも、隆弘くんの肩に……」

「気にしないで」

本当は、ずっとあのままでもよかったけど。

「あの、続きをやりましょう！」

美久が話題を変える。

「そうだね。じゃあ、実際に選択肢を作ってみよう」

僕たちは再びプログラミングに集中した。

でも、さっきまでの距離感が忘れられなくて、なんだかぎこちない。

◇◇◇◇

## 午後6時　今日の成果

「できた！」

画面には、簡単な選択肢ゲームが表示されている。

『主人公に告白する？』
［はい］［いいえ］

「はい」を選ぶと、「告白成功！ハッピーエンド」
「いいえ」を選ぶと、「また今度……」

シンプルだけど、ちゃんと動く。

「すごい！私が作ったゲームだ！」

美久が子供のように喜ぶ。

「まだ始まりだけどね」

「でも、嬉しい。隆弘くんのおかげです」

美久の笑顔を見て、僕も嬉しくなる。

「じゃあ、今日はここまで」

「はい。ありがとうございました」

片付けをしながら、美久が言う。

「あの、明日も……いいですか？」

「もちろん」

即答する僕。

「やった！じゃあ、明日はもっと複雑な分岐を作りましょう」

「楽しみだな」

◇◇◇◇

## 午後7時　美久を送る

「送るよ」

「え、でも隣の部屋……」

「いいから」

僕は美久を部屋の前まで送った。

「今日は楽しかったです」

「俺も」

「それじゃ、おやすみなさい」

「おやすみ」

ドアが閉まる前に、美久が振り返った。

「隆弘くん」

「ん？」

「私の好感度、上がりましたか？」

いたずらっぽく笑う美久。

「……カンストしそう」

僕の答えに、美久は真っ赤になって部屋に入っていった。

一人になった廊下で、僕は頭を抱えた。

（なんて答えてるんだ、俺は）

でも、嘘じゃない。

美久への好感度は、もう変数の最大値を超えそうだ。

明日も一緒にプログラミング。

このままいくと、僕たちの関係も、新しい分岐に進みそうな予感がした。
