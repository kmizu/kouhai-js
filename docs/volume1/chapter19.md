# 第19話 手を繋いで歩く週末

## 10月27日（金）午後5時

文化祭まで、あと一週間。

ゲームの完成度は80％くらい。

でも、最後の20％が、一番大変だって聞いてる。

「今週末、デバッグ作業しない？」

隆弘くんが提案してきた。

「うん、もちろん」

「土曜日、出かけない？」

思いがけない言葉に、心臓が跳ねる。

「え？」

「気分転換も大事だよ。ずっと部屋でコード見てても、効率悪いし」

隆弘くんの優しい気遣い。

でも、それだけじゃない気がする。

「どこに行くの？」

「美久が好きそうな場所」

謎めいた笑顔。

でも、その表情が好き。

◇◇◇◇

## 10月28日（土）午前10時　約束の場所

「おはよう」

待ち合わせ場所は、マンションのエントランス。

隆弘くんは、いつもと違って私服だった。

シンプルな白シャツに、黒いパンツ。

それだけなのに、すごくかっこいい。

「美久も、可愛いね」

私は、お気に入りのワンピースを着てきた。

淡いピンク色で、リボンがついてる。

「あ、ありがとう」

褒められて、顔が熱くなる。

「じゃあ、行こうか」

隆弘くんが手を差し出した。

「え？」

「人が多いから、はぐれないように」

そう言って、私の手を取る。

温かくて、大きな手。

ドキドキが止まらない。

◇◇◇◇

## 午前11時　プログラミング関連の書店

「ここ、新しくできた技術書専門店なんだ」

隆弘くんが連れてきてくれたのは、駅前の大きな書店。

プログラミング関連の本が、壁一面に並んでる。

「すごい！こんなにたくさん」

JavaScript、Python、Ruby、Go……

見たことのない言語の本もいっぱい。

「ゲーム開発の本もあるよ」

隆弘くんが指差した棚には、Unity、Unreal Engine、そしてノベルゲーム制作の本も。

「これ、欲しいかも」

手に取ったのは『JavaScriptで作るノベルゲーム入門』。

「買ってあげる」

「え、でも」

「デート代」

隆弘くんがさらっと言う。

デート。

その言葉に、心臓が高鳴る。

◇◇◇◇

## 午後0時30分　カフェでランチ

「美味しい」

書店の上にあるカフェで、パスタを食べる。

隆弘くんは、向かいの席で微笑んでる。

「本、ありがとう」

「気に入ってくれて良かった」

さっき買ってもらった本を、大事に抱える。

「ねえ、隆弘くん」

「ん？」

「どうしてプログラミング始めたの？」

前から聞きたかったこと。

隆弘くんは、少し考えてから答えた。

「中学の時、ゲームを作りたくて」

「ゲーム？」

「うん。好きな子のために」

その言葉に、胸がざわつく。

「その子は？」

「今、目の前にいる」

◇◇◇◇

## 午後1時　告白

「え？」

頭が真っ白になる。

「美久のために、ゲームを作りたかった」

隆弘くんが続ける。

「小学校の時、美久が『将来はゲームクリエイターになりたい』って言ってたの、覚えてる？」

そんなこと、言ったっけ。

でも、確かに言ったかもしれない。

「だから、プログラミング始めたんだ。美久と一緒にゲームを作れるように」

涙が出そうになる。

「ずっと、美久のこと見てた」

「隆弘くん……」

「今も、これからも、美久と一緒にいたい」

真っ直ぐな瞳。

嘘のない、本当の気持ち。

「私も」

声が震える。

「私も、隆弘くんとずっと一緒にいたい」

◇◇◇◇

## 午後2時　公園のベンチ

カフェを出て、近くの公園へ。

手を繋いだまま、ベンチに座る。

「正式に、付き合ってくれる？」

隆弘くんが改めて聞いてくる。

「はい」

即答だった。

もう迷いはない。

「良かった」

隆弘くんがほっとしたように笑う。

「実は、ゲームの告白シーン、美久への気持ちを込めて書いてたんだ」

「え？」

「だから、あんなに恥ずかしいセリフになっちゃって」

思い出す。

隆弘くんが書いた告白シーンのコード。

```javascript
// 告白シーン
function confession() {
    const dialogue = [
        "ずっと前から好きでした",
        "プログラミングを通じて、君の夢を一緒に叶えたい",
        "これからも、ずっと隣にいさせてください"
    ];
    
    return "永遠に、君のことを愛してる";
}
```

「あれ、私へのメッセージだったの？」

「うん」

隆弘くんが照れくさそうに頷く。

可愛い。

こんな隆弘くんも好き。

◇◇◇◇

## 午後3時30分　ゲームセンター

「久しぶりだね、ここ」

小学生の頃、よく来たゲームセンター。

「覚えてる？美久、クレーンゲーム下手だったよね」

「今は違うもん！」

ムキになって、クレーンゲームに挑戦。

でも、やっぱり取れない。

「貸して」

隆弘くんが代わってくれる。

一発で、ぬいぐるみをゲット。

「はい、プレゼント」

ピンクのウサギのぬいぐるみ。

「ありがとう」

大切に抱きしめる。

「名前、つけてあげて」

「うーん……ぴょん太！」

「可愛い名前」

二人で笑い合う。

幸せな時間。

◇◇◇◇

## 午後5時　夕暮れの川辺

「今日は楽しかった」

オレンジ色に染まる空を見上げる。

川面に映る夕日が、きらきら光ってる。

「美久」

「なに？」

「キスしていい？」

ストレートな質問に、顔が真っ赤になる。

でも、頷く。

隆弘くんが、そっと顔を近づけてくる。

目を閉じて、待つ。

柔らかい唇が、触れた。

初めてのキス。

甘くて、温かくて、幸せ。

離れた後、二人とも真っ赤になってる。

「好き」

「私も、大好き」

◇◇◇◇

## 午後6時30分　帰り道

手を繋いで、ゆっくり歩く。

もうすぐマンションに着いてしまう。

離れたくない。

「明日も会える？」

隆弘くんが聞く。

「うん」

「ゲームの続き、一緒に作ろう」

「はい」

でも、今はゲームのことより、隆弘くんとの時間が大切。

「美久」

「ん？」

「今日から恋人だね」

恋人。

その響きに、胸が熱くなる。

「うん、恋人」

エレベーターで7階に着く。

名残惜しそうに、手を離す。

「また明日」

「うん、また明日」

それぞれの部屋に入る前に、もう一度振り返る。

隆弘くんも、同じタイミングで振り返ってた。

目が合って、二人で笑う。

「おやすみ」

「おやすみなさい」

ドアを閉めて、背中にもたれかかる。

胸に抱いたぬいぐるみが、今日の幸せを物語ってる。

スマホが震えた。

『今日はありがとう。大好きだよ』

『私も大好き。明日も楽しみ』

『ゆっくり休んで。愛してる』

愛してる。

その言葉を、何度も読み返す。

幸せすぎて、眠れそうにない。

でも、明日も隆弘くんに会える。

それだけで、世界が輝いて見える。

私たちは、恋人になった。

プログラミングが繋いでくれた、運命の糸。

これからも、ずっと一緒に。

そう信じて、幸せな気持ちのまま、ベッドに入った。
